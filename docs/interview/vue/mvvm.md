# MVVM 模型

## 什么是 MVVM 模型

> MVVM 是 Model-View-ViewModel 的缩写。MVVM 本质上就是 MVC 模式的改进版。其中 Model 代表数据模型，View 代表 UI 即可以看到的界面，ViewModel 是 View 和 Model 的桥梁，数据会绑定到 ViewModel 层并自动将数据渲染到页面中，视图变化会自动更新数据,之前是通过操作 Dom 来更新视图，现在是数据驱动视图。
